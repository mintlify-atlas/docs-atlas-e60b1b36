## Overview

This endpoint returns all adoption records from the system. Each adoption links a user (owner) with an adopted pet.

## Endpoint

```
GET /api/adoptions
```

## Response

<ResponseField name="status" type="string" required>
  Status of the response. Returns "success" on successful request.
</ResponseField>

<ResponseField name="payload" type="array" required>
  Array of adoption objects.
  
  <ResponseField name="_id" type="string">
    Unique identifier for the adoption record.
  </ResponseField>
  
  <ResponseField name="owner" type="string | object">
    Reference to the user who adopted the pet. May be populated with full user object.
  </ResponseField>
  
  <ResponseField name="pet" type="string | object">
    Reference to the adopted pet. May be populated with full pet object.
  </ResponseField>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl --location 'http://localhost:8080/api/adoptions' \
--header 'Content-Type: application/json'
```

```javascript JavaScript
fetch('http://localhost:8080/api/adoptions', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
})
  .then(response => response.json())
  .then(data => console.log(data));
```

```python Python
import requests

url = "http://localhost:8080/api/adoptions"
headers = {"Content-Type": "application/json"}

response = requests.get(url, headers=headers)
print(response.json())
```

</CodeGroup>
