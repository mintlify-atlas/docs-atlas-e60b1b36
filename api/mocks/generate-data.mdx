This endpoint generates mock users and pets with realistic test data using Faker.js and inserts them directly into MongoDB. Generated pets can be randomly assigned to created users if they are marked as adopted.

## Query Parameters

<ParamField query="user" type="number" default="1">
  Number of mock users to generate and insert into the database. Must be a positive integer.
</ParamField>

<ParamField query="pet" type="number" default="1">
  Number of mock pets to generate and insert into the database. Must be a positive integer.
</ParamField>

## How It Works

1. **Generate Users**: Creates the specified number of mock users with Faker.js data:
   - Random first and last names
   - Email addresses based on the generated names (@gmail.com)
   - Hashed password (default: 'coder123')
   - Random role ('user' or 'admin')
   - Empty pets array

2. **Insert Users**: Batch inserts all generated users into MongoDB

3. **Generate Pets**: Creates the specified number of mock pets with:
   - Random pet name
   - Random animal species
   - Random birthdate
   - Random adoption status (boolean)
   - Random avatar image

4. **Handle Adoptions**: If a pet is generated with `adopted: true`, it's automatically assigned to a random user from the newly created users

5. **Insert Pets**: Batch inserts all generated pets into MongoDB

## Error Cases

- **400 Bad Request**: Returned when `user` or `pet` query parameters are negative values
- **500 Internal Server Error**: Returned when database insertion fails or other server errors occur

## Implementation Details

This endpoint uses the **Faker.js** library (Spanish Mexico locale) to generate realistic test data:

- **User data**: Generated by `generaUser()` function in `/workspace/source/src/mocks/mocks.js:29`
- **Pet data**: Generated by `generaPet()` function in `/workspace/source/src/mocks/mocks.js:4`
- **Database models**: Uses `userModel` and `petModel` for MongoDB operations
- **Batch insertion**: Uses MongoDB's `insertMany()` for efficient bulk inserts
- **Smart adoptions**: Adopted pets are randomly assigned to one of the newly created users
