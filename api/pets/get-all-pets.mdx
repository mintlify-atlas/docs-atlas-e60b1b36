---
title: 'Get All Pets'
api: 'GET /api/pets'
description: 'Retrieve a list of all pets in the system'
---

This endpoint returns all pets registered in the database, including their details such as name, species, birth date, adoption status, and associated images.

## Response

<ResponseField name="status" type="string" required>
  Status of the response. Returns "success" on successful retrieval.
</ResponseField>

<ResponseField name="payload" type="array" required>
  Array of pet objects.
  
  <ResponseField name="_id" type="string">
    Unique identifier for the pet
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Name of the pet
  </ResponseField>
  
  <ResponseField name="specie" type="string">
    Species of the pet (e.g., dog, cat, bird)
  </ResponseField>
  
  <ResponseField name="birthDate" type="string">
    Birth date of the pet in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="adopted" type="boolean">
    Indicates whether the pet has been adopted (default: false)
  </ResponseField>
  
  <ResponseField name="owner" type="string">
    User ID of the pet's owner (if adopted)
  </ResponseField>
  
  <ResponseField name="image" type="string">
    Path to the pet's image file
  </ResponseField>
</ResponseField>

## Example Request

```bash cURL
curl --request GET \
  --url http://localhost:8080/api/pets \
  --header 'Content-Type: application/json'
```
