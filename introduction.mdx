## Welcome to Adoptme API

Adoptme is a comprehensive backend API designed to facilitate pet adoption management. Built with Express.js and MongoDB, it provides a robust platform for managing users, pets, and adoption records with authentication, file uploads, and comprehensive documentation.

## Technology stack

Adoptme API is built with modern, production-ready technologies:

- **Express.js** - Fast, minimalist web framework
- **MongoDB** - Flexible NoSQL database with Mongoose ODM
- **JWT** - Secure token-based authentication
- **bcrypt** - Password hashing and encryption
- **Multer** - File upload handling for pet images
- **Swagger** - Interactive API documentation
- **Docker** - Containerized deployment support

## API overview

The Adoptme API is organized around REST principles with predictable resource-oriented URLs:

- **Base URL**: `http://localhost:8080` (configurable via PORT environment variable)
- **Authentication**: JWT tokens via cookies
- **Response format**: JSON
- **HTTP methods**: GET, POST, PUT, DELETE

### Main endpoints

- `/api/users` - User management and profiles
- `/api/pets` - Pet listings and management
- `/api/adoptions` - Adoption creation and tracking
- `/api/sessions` - Authentication and authorization
- `/api/mocks` - Test data generation
- `/docs` - Interactive Swagger documentation

The API includes Swagger UI documentation at `/docs` for interactive API exploration and testing.
